 # Movie App 🎬

A **React** movie application that allows users to **search for movies** and **view trending movies**. The app integrates **TMDB API** for show casing movie data



## Getting Started
### Clone the repository

```bash
git clone  https://github.com/minhaj-my/MovieMosaic-
cd movie-app

#Install dependencies
pnpm/bun install

#Create .env file
ITE_TMDB_API_KEY=YOUR_TMDB_API_KEY
 ```
